## Contributions welcome

Feel free to ask questions, enhancements and to contribute to this project!

## Development

Install `hatch`:

- for macOS: `brew install hatch`

Run the tests:

```bash
hatch run test:test
hatch run style:check
```

Serve the documentation:

```bash
hatch run doc:serve
```
