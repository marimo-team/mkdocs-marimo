## Contributions welcome

Feel free to ask questions, enhancements and to contribute to this project!

## Development

Install `pixi`: see [pixi docs](https://pixi.sh)

Run the tests:

```bash
pixi run postinstall
pixi run test
```

Serve the documentation:

```bash
pixi run docs
```
